이 코드는 오름차순의 두 리스트를 결합하는 함수입니다. 결합하여 만드는 새 리스트의 데이터 또한 오름차순입니다.
결과로, 더한 리스트 2개와 새로 생성된 리스트 1개를 출력해줍니다.
main에서 결합할 두 리스트 alist와 blist를 생성해주면 함수를 통해 clist에 새로운 리스트가 저장됩니다.

다른 리스트를 이용하고 싶다면 main함수에서 add_last(&alist, k); (단, k는 넣고 싶은 데이터)를 실행하면 됩니다. 단, 데이터는 오름차순으로 함수를 호출하여야 합니다.
예를 들어, alist에 10, 24, 35, 89 를 넣고 싶다면 add_last(&alist,10); add_last(&alist,24); add_list(&alist,35); add_list(&alist,89);를 main함수에서 실행하면 됩니다.
