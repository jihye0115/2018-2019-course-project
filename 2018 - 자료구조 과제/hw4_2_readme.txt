이 코드는 직원이 2명인 은행에서 10분 간 몇명의 손님이 오고, 평균 대기 시간은 몇분인지 출력하여 줍니다.

각 손님의 은행 도착 시간과 업무 시작 시간, 대기 시간, 업무하는데 걸린 총 시간을 출력합니다.
만약 시간당 손님이 도착하는 확률을 바꾸고 싶다면 arrival_prob를 새로 지정할 수 있습니다. 단, 값은 0에서 1사이의 확률을 대입해야 합니다.
또한, 더 긴 시간이나 짧은 시간 동안의 경향을 보고싶다면 durtaion을 새로 지정할 수 있습니다.
예) 30분 간의 경향을 보고 싶다면 int duration=30;을 실행하면 됩니다.
